Projects:

  • I built a command line version of the game Mastermind in Ruby. The player attempts to guess the correct color sequence while getting feedback from the computer each round. The game also includes an option to play against the computer when the player has the option to select the sequence and the computer makes informed guesses utilizing the feedback it received after each turn.

    View Github

  • The hotel project was an excercise in building a system using object oriented principles and TDD. The program runs within a command line interface and it was created with a TDD process where each test was designed based on the given user stories. This system allows a hotel administrator to make guest reservations and reserve blocks of rooms within a hotel.

    View Github

  • The RideShare project was built with a partner, where we worked through through the requirements in a TDD style, switching between navigator and driver rols to practice pair programming. The project involved reading in various CSV data sets and manipulating this data to store and dispatch drivers to various passengers requesting a ride.

    View Github
