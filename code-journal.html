First 6 weeks of Ada

We have been dedicating our time to learning Ruby. Practicing the use of arrays, hashes, loops for in class projects. Also covered the use of csv files as a means to organize small amounts of data. Followed by object orientated programming, becoming very familiar with classes and inheritance. Lots of debugging by understanding a stack trace, minitesting, code coverage and facilitating the use of a GuardFile to automate testing files. Covering test driven development with a few projects where we were given testing requirements, and we had to create working code that would get the tests to pass. Working on a few pair projects and using Github as a way to collaborate. Learning and using the terminal to interact with our computers--it saves time. Learning Agile methodologies and best practices when designing a program. CS Fundamentals have touched on: converting binary to decimals and vice versa, sets, arrays, time complexity, space complexity, creating algorithms for common built-in methods such as count, sum, min, max, average, median, binary search.
